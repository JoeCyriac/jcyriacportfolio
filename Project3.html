Project 3




Description:


Just like Project 2, this project was also done in my Web Development class during my Fall 2021 semester. I had 2 other teammates for this project. For this assignment, we used node.js to serve static files and dynamic routes for pages on energy consumption for a particular year, energy consumption for a particular state, and energy consumption for a particular energy source. There were 3 interesting facts we found while looking at the data that I analyzed here Something I learned while working on this project was how much more time-efficient and beneficial using a server to update a template rather than hard coding each page. Instead of having to update thousands of pages when an owner of a site wants to redesign, they can just update the template and fill in the data wherever they designate. To use this page, install node.js and npm install express and sqlite3.

Examples:


http://localhost:8000/year/2000 would return a page with each states consumption of each type of energy source, the total for each state, and an interactable graph that displays the percent of each resource used that year.

http://localhost:8000/state/MN would return a page with Minnesota's usage of each energy type every year from 1960-2018. It also displays an interactable graph to show the trends of usage throughout these years.

http://localhost:8000/energy/coal would return a page with each states usage of coal, for every year from 1960-2018. It also displays an interactable graph to show the trends of usage throughout these years for each state.
